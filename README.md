Η παρούσα πτυχιακή εργασία πραγματεύεται τον πλήρη κύκλο ανάπτυξης λογισμικού (Software Development Life Cycle - SDLC) για τη δημιουργία μιας διαδικτυακής εφαρμογής διαχείρισης εργασιών (Task Management System), ειδικά σχεδιασμένης για τις ανάγκες της φοιτητικής κοινότητας. Το κίνητρο για την υλοποίηση του έργου προήλθε από την παρατηρούμενη δυσκολία των φοιτητών να διαχειριστούν αποτελεσματικά τον χρόνο τους, γεγονός που συνδέεται βιβλιογραφικά με αυξημένα επίπεδα άγχους και χαμηλή ακαδημαϊκή επίδοση (Macan et al., 1990).
Τεχνολογικά, η εφαρμογή αναπτύχθηκε στη γλώσσα προγραμματισμού Java, αξιοποιώντας το πλαίσιο εργασίας Spring Boot για την υλοποίηση του Backend. Η αρχιτεκτονική που υιοθετήθηκε είναι η Model-View-Controller (MVC), η οποία επιτρέπει τον σαφή διαχωρισμό της επιχειρησιακής λογικής από τη διεπαφή χρήστη, ενισχύοντας τη συντηρησιμότητα του κώδικα (Fowler, 2002). Για τη διαχείριση και αποθήκευση των δεδομένων χρησιμοποιήθηκε το Σύστημα Διαχείρισης Σχεσιακών Βάσεων Δεδομένων (RDBMS) MySQL, ενώ η επικοινωνία με τη βάση επιτεύχθηκε μέσω του προτύπου JPA/Hibernate. Το Frontend υλοποιήθηκε με τη χρήση της μηχανής προτύπων Thymeleaf, προσφέροντας δυναμικό περιεχόμενο στον τελικό χρήστη (Hocking, 2015).
Η μεθοδολογία περιελάμβανε ανάλυση απαιτήσεων, σχεδιασμό συστήματος με διαγράμματα UML (Unified Modeling Language), υλοποίηση βάσει των αρχών του Clean Code και διεξοδικό έλεγχο λειτουργίας (Software Testing). Τα αποτελέσματα της εργασίας καταδεικνύουν ότι η χρήση σύγχρονων εργαλείων ανοικτού κώδικα μπορεί να οδηγήσει στην ταχεία ανάπτυξη ασφαλών και επεκτάσιμων εφαρμογών.
